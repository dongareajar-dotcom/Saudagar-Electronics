⚡ Saudagar Electronics – CI/CD Project
📌 Project Description

Saudagar Electronics is a simple electronic shop website built using HTML and CSS.
The main purpose of this project is to learn and demonstrate a real-world DevOps CI/CD pipeline using AWS services.

Whenever code is updated and pushed to GitHub, it is automatically deployed to a live website using AWS pipeline and S3 hosting.

🎯 Project Goal

The goal of this project is:

To create a simple business website
To learn Git & GitHub workflow
To understand CI/CD automation
To deploy a website on AWS cloud automatically
⚙️ How It Works (Simple Flow)
GitHub → AWS CodePipeline → AWS CodeBuild → AWS S3 → Live Website
🛠️ Technologies Used
HTML – Website structure
CSS – Styling and design
Git & GitHub – Version control
AWS CodePipeline – Automation of deployment
AWS CodeBuild – Build and validation
AWS S3 – Hosting the website
🚀 Features
Simple electronic shop website
Attractive UI design
Marathi + English content
Product display section
Fully automated deployment system
No manual upload required after setup
💡 What I Learned
How CI/CD pipeline works in real projects
How AWS services connect together
How Git and GitHub manage code versions
How automatic deployment saves time
How real DevOps systems work in companies
